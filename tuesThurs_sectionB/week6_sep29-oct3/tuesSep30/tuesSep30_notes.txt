Strings. 

String first = "hello";
String second = "goodbye";

How to print all the characters in first?

for (int i = 0; i < first.length(); i++) {
    System.out.println(first.charAt(i));
}




What is this doing?

for (int i = 0; i < second.length(); i++) {
    first += second.charAt(i);
}

first will be: "hellogoodbye"



...is there a better way? very inefficient

use StringBuilder!

StringBuilder build = new StringBuilder(first);
for (int i = 0; i < second.length(); i++) {
    build.append(second.charAt(i));
}

String result = build.toString();
